## Vendor Performance Analysis using Python , SQL and Power BI

## Overview
This project analyzes vendor sales data to understand how different vendors are performing.  
The main goal is to identify strong and weak vendors, observe sales trends, and generate useful insights that can help in better business decision-making.

The analysis is done using Python for data processing and Power BI for visualization.

---

## Project Goals
- Analyze vendor-wise sales performance  
- Identify top-performing and low-performing vendors  
- Understand sales trends and patterns  
- Create clear and useful visual reports  

---
